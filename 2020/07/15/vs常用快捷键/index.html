vs

在vs中设置或查看快捷键

在工具栏 工具 => 选项 => 环境 => 键盘 中查看或修改快捷键

常用快捷键

智能提示：ctrl + J
CTRL + SHIFT + B 生成解决方案
F4 显示属性窗口
F5 运行调试
F10 跨过程序执行
F11 单步逐句执行
F7: 查看代码
Shift+F7: 查看窗体设计器
Ctrl+F: 查找
F3: 查找下一个
Ctrl+H: 替换
Ctrl+F5: 开始执行(不调试)
F9: 切换断点
Ctrl+Shift+F9: 删除全部断点
Ctrl+Shift+F: 在文件中查找
Ctrl+K,S: 插入外侧代码

代码格式化：ctrl+E+F

删除当前行 Ctrl+Shift+L

注释 Ctrl + K + C    取消注释 Ctrl + K + U
